## Json som DB?
En Json-baseret database er smart fordi det er en simpel og fleksibel måde at opbevare data, man behøver ikke at oprette og administrere en reel sql database. Det er nemt at versionere "databasen" over git, da en Json-baseret database jo bare er en tekstfil.
